01Install the skill

Keep the tool local.

Install once for Claude Code, Codex, or opencode. The packaged renderers and validators work without a hosted diagram service.

Permanent install
npx skills add tt-a1i/archify -g
One-off Codex run
npx skills use tt-a1i/archify@archify --agent codex

The viewer link carries only the diagram type. It does not include the title, nodes, edges, repository path, or source JSON.
